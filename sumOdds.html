Adding Odd Integers

©2020 • Kristin Bunnell • BYU-Idaho • CIT 160
write a program that asks the user for an integer n and then outputs the sum of all the odd integers between 1 and n, inclusive. For example, if a user entered 10, your program would add 1 + 3 + 5 + 7 + 9 and output 25. If a user entered 11, your program would add 1 + 3 + 5 + 7 + 9 + 11 and output 36.
